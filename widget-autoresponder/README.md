# AutoResponder - Bot de Objeciones para Pasajeros

Bot inteligente para generar respuestas automáticas a objeciones de pasajeros usando IA.

## ¿Cómo funciona?

1. Copia la objeción del pasajero en WhatsApp
2. Pégala en el campo de entrada
3. Haz clic en "Generar Respuesta"
4. Copia la respuesta generada
5. Pégala en WhatsApp

## Features

✨ Respuestas automáticas con IA
📋 Historial de últimas respuestas
📋 Copiar al portapapeles con un clic
⚡ Interfaz rápida y moderna
💬 Optimizado para WhatsApp
